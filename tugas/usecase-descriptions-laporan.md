# Use Case Descriptions - Laporan & Analytics

## Login
Owner/Admin/Cashier login ke sistem laporan.

## Pilih Jenis Laporan
Pengguna memilih jenis laporan yang ingin dilihat.

## Set Filter
Pengguna mengatur filter (periode, cabang, dsb).

## Lihat Laporan
Pengguna melihat hasil laporan dan analitik.

## Export/Print
Pengguna dapat mengekspor atau mencetak laporan.
