# Use Case Descriptions - Pengeluaran Cabang

## Login
Admin login ke sistem pengeluaran cabang.

## Tambah Pengeluaran
Admin menambah data pengeluaran baru.

## Edit Pengeluaran
Admin mengedit data pengeluaran yang sudah ada.

## Hapus Pengeluaran
Admin menghapus data pengeluaran yang tidak diperlukan.

## Upload Bukti
Admin mengunggah bukti pengeluaran (nota, foto, dll).

## Lihat Riwayat
Admin melihat riwayat pengeluaran cabang.
