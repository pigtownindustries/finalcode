# Use Case Descriptions - POS System

## Login
Cashier melakukan login ke sistem POS untuk mengakses fitur transaksi.

## Buat Transaksi
Cashier memulai transaksi baru untuk pelanggan.

## Tambah Item
Cashier menambahkan layanan atau produk ke dalam transaksi.

## Pilih Karyawan
Cashier memilih karyawan/barber yang melayani pelanggan.

## Input Diskon
Cashier dapat memasukkan diskon jika ada promo atau potongan harga.

## Pilih Metode Bayar
Cashier memilih metode pembayaran (cash, debit, QRIS, dll).

## Cetak Struk
Cashier mencetak struk transaksi untuk pelanggan.
