# Use Case Descriptions - Presensi Karyawan

## Login
Karyawan/Admin login ke sistem presensi.

## Ambil Foto
Karyawan mengambil foto selfie sebagai bukti kehadiran.

## Input Lokasi
Karyawan menginput lokasi saat presensi.

## Submit Presensi
Karyawan mengirim data presensi ke sistem.

## Validasi Presensi
Admin memvalidasi data presensi yang masuk.

## Lihat Riwayat
Karyawan dapat melihat riwayat presensi mereka.
