# Use Case Descriptions - Dashboard Owner

## Login
Owner login ke dashboard owner.

## Lihat Metrik Bisnis
Owner melihat metrik bisnis utama (revenue, transaksi, pengeluaran, dsb).

## Lihat Performa Cabang
Owner melihat performa masing-masing cabang.

## Lihat Top Karyawan
Owner melihat karyawan dengan performa terbaik.

## Akses Menu Lain
Owner mengakses menu lain dari dashboard (laporan, gaji, cabang, karyawan, settings).
